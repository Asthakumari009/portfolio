# SAAD° Portfolio

A single-page portfolio site built with HTML, CSS, and JavaScript, featuring a Three.js background scene and custom UI interactions.

## Structure

- index.html: Layout and content
- styles.css: Styling and animations
- main.js: Three.js scene
- app.js: UI interactions and effects
- serve.bat: Local dev server helper

## Run locally

On Windows, run:

```
serve.bat
```

This starts a static server on http://localhost:5173.

## Deploy to Vercel

1. Import this repo in Vercel.
2. Framework Preset: Other.
3. Build Command: leave empty.
4. Output Directory: .
5. Deploy.

The routing behavior is handled by vercel.json.
